 PDF转Word工具

一个简单实用的PDF转Word转换工具，基于Python开发，提供直观的图形化界面，帮助用户轻松将PDF文档转换为可编辑的Word文档。

## 功能特点

- 简单易用的图形化界面
- 支持选择任意PDF文件进行转换
- 自动生成Word文件保存路径
- 支持自定义输出路径
- 转换进度实时反馈
- 完善的错误处理机制

## 安装说明

### 环境要求

- Python 3.6 或更高版本

### 依赖库

本项目需要安装以下依赖库：

```
pip install pdf2docx
```

## 使用方法

1. 运行程序：
   ```
   python pdf转换word.py
   ```

2. 在程序界面中：
   - 点击"浏览"按钮选择需要转换的PDF文件
   - 系统会自动生成对应的Word文件保存路径，您也可以通过"浏览"按钮自定义保存位置
   - 点击"转换"按钮开始转换
   - 等待转换完成，系统会弹出提示框告知转换结果

## 注意事项

- 转换效果取决于原始PDF文件的质量和格式
- 对于扫描版PDF（图片格式），可能无法直接转换文字内容
- 转换大文件可能需要较长时间，请耐心等待

## 技术实现

- **PDF转Word**：使用pdf2docx库实现转换功能
- **图形界面**：使用Python内置的Tkinter库构建用户界面
- **错误处理**：实现了完善的异常捕获和用户提示机制
